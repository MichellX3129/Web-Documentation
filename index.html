    Designs

    First a critical look at the existing website was take. The existing website has a lack of information displayed. The landing page is short, in order to obtain key information’s such as how to help, key insights and goals are not immediately apparent in the landing page.

    Therefore the design goal is to imporve on those aspects as well as to affirm to the user needs listed on the User profiles.

    Sketches

    In the first stage of sketching, the navigation bar is place all the way at the top of the webpage. And the large strip of image will draw attention along side with the Donateprompt. Both logos are listed allows the user to understand the 2 departments of the charity.

    In this second ideation of sketching, more emphasis is place on the 2 departments . They would have 2 different call the action prompts on on each side. Emphaising composition of the organisation's structure. 300 Blankets being predominatly function by donations to fund the supply destribution and the Soul Kitchen being more community/volunteering oriented. Having the 2 main ways for sustainability outlined at the top enabled the user to see it immedaitely. Following it, would be oulined as well as the chartity's introduction videos, allowing the user understand key insights of the organisation.

    The other main emphasis of the this sketch proposed website is the introduction of the community of the charity. This would enable the user to click into the other links to learn more about the overall community.
